This page demonstrates the new enhanced code snippet component with macOS-style headers, optional line numbers, and copy functionality. **No imports needed!** - The component is configured globally.

## Basic Code Block

Here's a simple JavaScript example:

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

## Code Block with Filename

You can add a filename to the header using the `filename` metadata:

```javascript filename=app.js
import express from 'express';

const app = express();
const PORT = process.env.PORT || 3000;

app.get('/', (req, res) => {
  res.json({ message: 'Hello, World!' });
});

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});
```

## Code Block without Line Numbers

You can disable line numbers by setting `lines=false`:

```python lines=false
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```

## Code Block with Line Highlighting

You can highlight specific lines using the `{1,3-5}` syntax:

```typescript filename=user.service.ts {2,4-6}
import { Injectable } from '@nestjs/common';
import { User } from './user.entity';

@Injectable()
export class UserService {
  private users: User[] = [];

  async findAll(): Promise<User[]> {
    return this.users;
  }

  async findById(id: string): Promise<User | null> {
    return this.users.find(user => user.id === id) || null;
  }
}
```

## Features

The enhanced code snippet component includes:

- **macOS-style header** with colored dots
- **Filename display** when specified in metadata
- **Language indicator** when no filename is provided
- **Copy to clipboard** functionality with visual feedback
- **Optional line numbers** (enabled by default)
- **Line highlighting** support using Shiki's syntax
- **Dark/light theme** support
- **Responsive design** with horizontal scrolling for long lines

## Usage

The enhanced code snippet component is **automatically enabled** for all MDX files! No imports needed.

To use these features in your MDX files:

1. **Just create `.mdx` files** - No imports required!

2. **Add metadata** to your code blocks:
   - `filename=example.js` - Shows filename in header
   - `lines=false` - Disables line numbers
   - `{1,3-5}` - Highlights specific lines

3. **Enjoy enhanced code blocks** with all the new features!

Example usage:
```javascript filename=app.js {2,4}
import express from 'express';
const app = express(); // highlighted
const PORT = 3000;
app.listen(PORT); // highlighted
```
